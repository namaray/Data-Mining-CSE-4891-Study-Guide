Activation Functions & Stochastic Models

An Activation Function decides whether a neuron should be activated (fire) or not based on the weighted sum of inputs.

The Common Example

For all examples below, we use these inputs and weights from the notes:

Inputs: I₁ = 1, I₂ = 1
Weights: w₁ = 0.5, w₂ = -0.3
Induced Local Field (v): The weighted sum.
v = (I₁ * w₁) + (I₂ * w₂)
v = (1 * 0.5) + (1 * -0.3)
v = 0.5 - 0.3
v = 0.2

We will see how different functions handle this input of 0.2.

1. Threshold Function

Also known as the Step Function. It is binary (0 or 1).

Rule:
If v ≥ 0 → Output = 1
If v < 0 → Output = 0

Calculation:

Since v = 0.2
And 0.2 ≥ 0
Output = 1

2. Piecewise-Linear Function

A linear slope between -0.5 and +0.5, and flat (saturated) outside that range.

Rule:
1. If v ≥ 0.5 → Output = 1
2. If -0.5 < v < 0.5 → Output = v + 0.5
3. If v ≤ -0.5 → Output = 0

Calculation:

Since v = 0.2
Check range: -0.5 < 0.2 < 0.5 (It is in the middle)
Output = 0.2 + 0.5
Output = 0.7

3. Sigmoid Function (Logistic)

An "S" shaped curve. It maps any value to a number between 0 and 1.

Formula: φ(v) = 1 / (1 + e⁻ᵃᵛ)
where 'a' is the slope parameter. (We assume a=1 for this example).

Calculation:

v = 0.2
φ(0.2) = 1 / (1 + e⁻⁰˙²)
φ(0.2) = 1 / (1 + 0.8187)
φ(0.2) = 1 / 1.8187
Output = 0.5498

4. Signum Function (Hyperbolic Tangent)

Uses the `tanh` function. Maps values between -1 and +1.

Formula: φ(v) = tanh(v)

Calculation:

v = 0.2
φ(0.2) = tanh(0.2)
Output = 0.1973

5. Stochastic Model of a Neuron

In this model, the neuron doesn't have a fixed output. Instead, it has a Probability of firing (+1) or resting (-1).

Probability of Firing P(v):
P(v) = 1 / (1 + e⁻ᵛ/ᵀ)
T = Pseudo-temperature (Controls noise/uncertainty).

Effect of T (Temperature):

  • T → 0: Deterministic (Noiseless). The neuron behaves exactly like a Threshold function.
  • T → ∞: Pure Noise. The probability becomes 0.5 (random guess).

Calculation Example (High Noise)

Let v = 0.2 and T = 10,000.

Exponent = -v / T
Exponent = -0.2 / 10000 = -0.00002

P(0.2) = 1 / (1 + e⁻⁰˙⁰⁰⁰⁰²)
P(0.2) = 1 / (1 + 0.99998)
P(0.2) = 1 / 1.99998
Firing Probability ≈ 0.500005 (Almost 50/50 random)
