A step-by-step guide to updating weights using the Sigmoid Activation Function and Backpropagation.

1. Problem Setup & Visual Model

Initial Weights & Parameters

  • w1: 0.5
  • w2: 0.3
  • Bias (b): 0.1
  • Learning Rate (η): 0.1
  • a: 1 (Slope parameter)

Test Data

Input: (0, 1)

Key Formulas:
1. Sum (v) = I₁w₁ + I₂w₂ + b
2. Activation (f) = 1 / (1 + e⁻ᵃᵛ)
3. Error Signal (δ) = f(1-f)(d-f)
4. Weight Update (Δw) = η * Input * δ

2. Training Pattern 1: (1, 1, Target=1)

Step A: Forward Pass

Calculate the actual output with current weights.

v = (1 * 0.5) + (1 * 0.3) + 0.1
v = 0.9

f = 1 / (1 + e⁻⁰˙⁹)
f = 1 / (1 + 0.406)
f = 0.711

Step B: Backpropagation (Error Calculation)

Calculate how much the weights need to change (δ).

δ = f * (1 - f) * (d - f)
δ = 0.711 * (1 - 0.711) * (1 - 0.711)
δ = 0.711 * 0.289 * 0.289
δ = 0.05938

Step C: Weight Updates

New Weight = Old Weight + (η * Input * δ)

Weight Calculation (η * Input * δ) Δ Change New Value
w1 0.1 * 1 * 0.05938 0.005938 0.5059
w2 0.1 * 1 * 0.05938 0.005938 0.3059
bias 0.1 * 1 * 0.05938 0.005938 0.1059

3. Training Pattern 2: (0, 0, Target=0)

Note: We use the NEW weights calculated above.

Step A: Forward Pass

v = (0 * 0.5059) + (0 * 0.3059) + 0.1059
v = 0.1059

f = 1 / (1 + e⁻⁰˙¹⁰⁵⁹)
f = 1 / (1 + 0.8995)
f = 0.526

Step B: Backpropagation

δ = f * (1 - f) * (d - f)
δ = 0.526 * (1 - 0.526) * (0 - 0.526)
δ = 0.526 * 0.474 * (-0.526)
δ = -0.1311

Step C: Weight Updates

⚠️ Important: Since Inputs I1 and I2 are 0, the weights w1 and w2 will NOT change.

Weight Calculation (η * Input * δ) Δ Change New Value
w1 0.1 * 0 * -0.1311 0 0.5059 (No Change)
w2 0.1 * 0 * -0.1311 0 0.3059 (No Change)
bias 0.1 * 1 * -0.1311 -0.01311 0.09279
