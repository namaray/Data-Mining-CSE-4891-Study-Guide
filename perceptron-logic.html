Perceptron Logic Gates (Full Step-by-Step)

This page shows the complete manual calculation logs for training a Perceptron, exactly as done in the lecture notes.

üß† The Learning Rules

  1. Sum Calculation: $Sum = (I_1 \times w_1) + (I_2 \times w_2) + \dots$
  2. Output Check: If $Sum > Threshold$, Output is 1. Else 0.
  3. Weight Update:
    • If Expected 1 but got 0 (Too Low): Add Learning Rate to active weights.
    • If Expected 0 but got 1 (Too High): Subtract Learning Rate from active weights.
    • If Output == Expected: Do nothing.
