Perceptron Logic Gates

Complete step-by-step weight update calculations

The Algorithm Used:
1. Sum: $\sum (Input \times Weight)$
2. Check: If Sum > Threshold → Output 1. Otherwise → Output 0.
3. Update: $W_{new} = W_{old} + \alpha \times (Expected - Actual) \times Input$
(In simpler terms: If output is too low, Add Rate. If too high, Subtract Rate.)
