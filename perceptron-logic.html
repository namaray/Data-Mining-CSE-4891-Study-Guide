Perceptron Logic Gates (OR & AND)

This guide explains how a single neuron "learns" to become a logic gate by adjusting its weights based on errors.

üß† The Logic Rules (Plain English)

  1. Calculate Sum: Multiply Inputs by Weights and add them up.
  2. Check Threshold:
      • If Sum > Threshold → Output is 1.
      • Otherwise → Output is 0.
  3. Compare with Target:
      • Match? Do nothing.
      • False Negative (Expected 1, got 0)? The sum was too low. INCREASE the weights of active inputs.
      • False Positive (Expected 0, got 1)? The sum was too high. DECREASE the weights of active inputs.

*Active Input: An input that is currently 1. If an input is 0, changing its weight won't change the sum, so we leave it alone.

Problem 1: OR Operation

Goal: Output 1 if either input is 1.

Setup: Threshold = 0.9 | Learning Rate (c) = 0.2 | Initial Weights: w1=0.7, w2=0.7

Iteration 1

I1I2 w1w2 Sum Out Exp Action & Explanation
11 0.70.7 1.411 Do nothing.
1.4 > 0.9. Correct.
10 0.70.7 0.701 Increase w1.
Sum (0.7) too low. Target is 1.
New w1 = 0.7 + 0.2 = 0.9
01 0.90.7 0.701 Increase w2.
Sum (0.7) too low. Target is 1.
New w2 = 0.7 + 0.2 = 0.9

...After iteration 2, the weights stabilize at 0.9, 0.9.

Problem 2: AND Operation (2 Inputs)

Goal: Output 1 only if BOTH inputs are 1.

Setup: Threshold = 0.7 | Learning Rate (e) = 0.3 | Initial Weights: w1=0.8, w2=0.8

Iteration 1

I1I2 w1w2 Sum Out Exp Action & Explanation
11 0.80.8 1.611 Do nothing.
10 0.80.8 0.810 Decrease w1.
Sum (0.8) too high. Target 0.
New w1 = 0.8 - 0.3 = 0.5
01 0.50.8 0.810 Decrease w2.
Sum (0.8) too high. Target 0.
New w2 = 0.8 - 0.3 = 0.5

Problem 3: AND Operation (3 Inputs)

Goal: Output 1 only if I1, I2, AND I3 are all 1.

Setup: Threshold = 0.9 | Learning Rate (e) = 0.4 | Initial Weights: w1=0.8, w2=0.8, w3=0.8

Iteration 1

I1I2I3 w1w2w3 Sum Out Exp Action & Explanation
111 0.80.80.8 2.411 Do nothing.
110 0.80.80.8 1.610 Decrease Active w by 0.4.
Sum (1.6) > 0.9. Target 0.
Inputs I1 and I2 are active.
New w1 = 0.4, New w2 = 0.4.
101 0.40.40.8 1.210 Decrease Active w by 0.4.
Sum (0.4+0+0.8 = 1.2) > 0.9.
Target 0. Decrease I1 and I3.
New w1 = 0, New w3 = 0.4.

Final Stable State

After several iterations, the network learns the pattern. (Page 7 of notes).
