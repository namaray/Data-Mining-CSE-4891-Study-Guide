Perceptron Logic Gates (OR & AND)

How to manually calculate the iterations to train a simple neural network.

üß† The Learning Rules (from your notes)

Activation (Act): Sum of (Input √ó Weight). If Act > Threshold, Output is 1. Else 0.

Case 1: Output matches Expected?
→ Do Nothing.

Case 2: Output is 0, Expected is 1 (False Negative)?
→ INCREASE active weights by Learning Rate (c).

Case 3: Output is 1, Expected is 0 (False Positive)?
→ DECREASE active weights by Learning Rate (c).

*Active Weight: A weight connected to an input that is 1. If Input is 0, we do not update that weight.
